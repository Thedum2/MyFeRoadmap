# 월드 와이드 웹

#개념

월드 와이드 웹은 인터넷 위에서 문서를 연결하고 탐색하게 만든 정보 시스템이다. 핵심은 문서 하나를 고립된 파일로 두지 않고, 링크를 통해 다른 문서와 연결된 그래프로 다루는 데 있다.

초기 배경은 단순했다. 네트워크와 컴퓨터는 이미 있었지만 정보가 시스템마다 흩어져 있어 찾기 어려웠다. 그래서 URL로 자원을 식별하고, HTTP로 자원을 전달하며, HTML로 문서 구조를 표현하는 공통 규약이 만들어졌다.

![[assets/images/웹-클라이언트서버-모델.svg]]

low-level에서는 브라우저가 URL을 해석해 HTTP 요청을 보내고, 서버가 HTML 같은 자원을 응답한다. 브라우저는 응답을 파싱해 화면에 렌더링한다. 이 단순한 반복이 웹의 기본 동작이다.

한계도 있다. 문서 중심으로 시작했기 때문에 실시간 상호작용, 상태 관리, 복잡한 UI 같은 요구는 나중에 추가 계층이 필요했다. 그래서 CSS, JavaScript, 렌더링 엔진 기술이 발전하며 웹이 애플리케이션 플랫폼으로 확장됐다.
