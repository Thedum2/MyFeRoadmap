# 가상화

#개념

가상화는 하나의 물리 자원을 여러 개의 논리 자원처럼 분할해 사용하는 기술이다. 서버, 스토리지, 네트워크를 유연하게 할당할 수 있게 만든다.

클라우드와 현대 데이터센터의 핵심 전제는 가상화다. 물리 장비를 직접 고정 배치하는 방식보다 자원 활용률과 운영 민첩성이 높다.

low-level에서는 하이퍼바이저가 CPU, 메모리, I/O 접근을 중재해 가상 머신을 격리한다. 각 VM은 독립 운영체제처럼 보이지만 실제로는 공통 하드웨어를 공유한다.

가상화의 한계는 오버헤드와 운영 복잡도다. 고성능 워크로드에서는 베어메탈이 유리할 수 있고, 대규모 환경에서는 자동화가 필수다.
