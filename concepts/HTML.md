# HTML

#개념

HTML은 웹 문서의 구조와 의미를 기술하는 마크업 언어다. 제목, 문단, 목록, 링크 같은 요소를 태그로 표현해 브라우저가 문서 구조를 일관되게 해석하도록 만든다.

핵심 설계는 구조와 의미 중심이다. 어떻게 보여줄지는 CSS로, 어떻게 동작할지는 JavaScript로 분리한다. 이 분리 덕분에 문서 재사용성과 접근성이 높아진다.

![[assets/images/HTML-DOM-트리.svg]]

low-level에서는 HTML 문자열이 토큰화되고 파싱돼 DOM 트리로 변환된다. DOM은 렌더링과 스크립트 조작의 기준 데이터 구조가 된다.

한계는 HTML만으로 복잡한 표현과 동작을 다루기 어렵다는 점이다. 그래서 CSSOM, 렌더링 엔진, JavaScript 실행 모델이 결합된 현대 웹 스택이 만들어졌다.
