# 하이퍼바이저

#개념

하이퍼바이저는 물리 하드웨어 위에서 가상 머신을 생성하고 격리하는 가상화 계층 소프트웨어다. 여러 운영체제가 한 장비에서 독립적으로 동작하도록 만든다.

등장 배경은 자원 활용률 개선이다. 서버 한 대에 서비스 하나만 올리는 방식은 낭비가 크다. 하이퍼바이저는 하드웨어를 공유 자원으로 바꿔 활용률을 높인다.

low-level에서는 CPU 스케줄링, 메모리 매핑, 가상 장치 에뮬레이션을 담당한다. 게스트 OS는 실제 하드웨어를 직접 제어하지 않고 하이퍼바이저를 통해 접근한다.

한계는 추상화 계층에서 생기는 성능 비용과 운영 복잡성이다. 그래서 워크로드 특성에 따라 VM과 컨테이너를 혼합하는 설계가 자주 쓰인다.
