# PaaS

#개념

PaaS는 애플리케이션 실행 플랫폼을 관리형으로 제공하는 모델이다. 인프라 관리 대신 코드 배포와 런타임 운영에 집중할 수 있게 해준다.

IaaS보다 추상화가 높아 배포 속도가 빠르다. 개발팀은 OS 패치, 런타임 설치, 스케일링 기초 작업에서 벗어나 제품 기능에 집중할 수 있다.

low-level에서는 플랫폼이 빌드, 배포, 오토스케일, 모니터링 일부를 자동 처리한다. 사용자는 애플리케이션 코드와 환경 설정 중심으로 운영한다.

한계는 플랫폼 제약과 이식성 문제다. 특정 서비스에 맞춘 설계가 누적되면 벤더 락인 위험이 커질 수 있다.
