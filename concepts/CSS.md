# CSS

#개념

CSS는 웹 문서의 표현 규칙을 정의하는 스타일 언어다. HTML이 문서 구조를 담당한다면, CSS는 색상, 배치, 여백, 반응형 동작 같은 시각 표현을 담당한다.

CSS가 등장한 이유는 구조와 표현의 분리다. 표현을 HTML에 섞어 쓰면 유지보수 비용이 급격히 커진다. 스타일을 별도 계층으로 분리하면 변경 범위를 줄이고 재사용성을 높일 수 있다.

![[assets/images/CSS-박스모델.svg]]

low-level에서는 브라우저가 CSS를 파싱해 CSSOM을 만들고, DOM과 결합해 렌더 트리를 계산한다. 이후 레이아웃과 페인트를 수행해 최종 화면을 만든다.

한계는 cascade와 specificity의 복잡성이다. 규모가 커질수록 규칙 충돌이 늘어나므로 구조화 방법론과 설계 기준이 필요해진다.
