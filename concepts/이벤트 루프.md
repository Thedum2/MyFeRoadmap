# 이벤트 루프

#개념

이벤트 루프는 JavaScript 실행 환경에서 비동기 작업을 조율하는 스케줄링 모델이다. 한 줄 요약하면 "지금 실행할 일"과 "나중에 실행할 일"을 안전하게 교대시키는 메커니즘이다.

JavaScript는 단일 스레드 실행 모델을 기본으로 하므로 긴 작업이 메인 흐름을 막으면 UI가 멈춘다. 이벤트 루프는 큐 기반으로 작업을 분산해 반응성을 유지한다.

low-level에서는 콜 스택이 비면 태스크 큐에서 콜백을 꺼내 실행한다. 브라우저 API, 타이머, 네트워크 완료 이벤트가 큐에 들어오고 루프가 이를 순환 처리한다.

이 구조 덕분에 동시성처럼 보이는 동작이 가능하지만, 실제로는 시간 분할 실행에 가깝다. 그래서 블로킹 연산을 줄이는 설계가 성능의 핵심이 된다.
