# 렌더링 엔진

#개념

렌더링 엔진은 HTML/CSS를 해석해 최종 화면을 계산하는 브라우저 내부 모듈이다. 문서 텍스트를 실제 픽셀로 바꾸는 변환기라고 보면 된다.

핵심 흐름은 파싱, 트리 구성, 레이아웃, 페인트다. HTML은 DOM으로, CSS는 CSSOM으로 바뀌고, 두 트리가 결합돼 렌더 트리가 만들어진다. 이후 각 요소의 크기와 위치를 계산해 화면에 그린다.

low-level에서 성능 병목은 레이아웃 재계산과 페인트 재실행에서 자주 발생한다. 그래서 어떤 속성을 바꾸는지, 얼마나 자주 바꾸는지가 UX 품질에 직접 연결된다.

렌더링 엔진은 표준 구현 차이 때문에 브라우저 간 미묘한 동작 차이를 만들기도 한다. 이 차이를 줄이기 위해 표준과 호환성 테스트가 계속 축적된다.
