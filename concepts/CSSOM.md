# CSSOM

#개념

CSSOM은 브라우저가 CSS 규칙을 메모리에서 다루기 위해 구성한 객체 모델이다. HTML의 DOM과 짝을 이루는 스타일 트리라고 보면 된다.

브라우저는 CSS 텍스트를 그대로 쓰지 않는다. 선택자, 상속, 우선순위를 계산하기 위해 규칙을 구조화된 객체로 변환한다. 이 구조가 CSSOM이다.

low-level에서는 DOM 노드마다 어떤 규칙이 적용되는지 매칭하고, 계산된 스타일을 바탕으로 렌더 트리를 구성한다. 스타일 변경은 CSSOM 재평가와 후속 레이아웃 연산으로 이어진다.

CSSOM이 중요한 이유는 스타일 변경이 성능에 직접 영향을 주기 때문이다. 어떤 속성을 자주 바꾸는지가 렌더링 비용을 결정한다.
