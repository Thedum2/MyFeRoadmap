# JavaScript

#개념

JavaScript는 브라우저에서 실행되는 스크립트 언어로, 웹 페이지를 정적 문서에서 상호작용 애플리케이션으로 바꾼 핵심 기술이다.

등장 배경은 명확했다. 사용자 입력에 즉시 반응하려면 서버 왕복만으로는 한계가 컸다. 브라우저 내부에서 이벤트를 처리하고 DOM을 직접 조작할 수 있어야 했다.

![[assets/images/JavaScript-이벤트루프.png]]

low-level에서는 이벤트 루프와 콜백 큐를 통해 비동기 작업이 스케줄링된다. 네트워크 응답, 타이머, 사용자 입력 같은 이벤트가 큐에 쌓이고 실행 컨텍스트가 순차 처리된다.

한계는 복잡도 증가다. 상태 관리, 비동기 오류 처리, 번들 크기, 보안 이슈가 함께 커진다. 그래서 타입 시스템, 모듈화, 성능 최적화 전략이 중요해진다.
