# Infrastructure as Code

#개념

Infrastructure as Code는 인프라 구성을 코드로 선언하고 자동으로 배포·변경하는 운영 방식이다. 수동 클릭 기반 운영을 재현 가능한 배포 파이프라인으로 바꾼다.

수동 설정은 환경 불일치와 휴먼 에러를 만들기 쉽다. IaC는 인프라 상태를 코드로 버전 관리해 변경 이력과 롤백 가능성을 확보한다.

low-level에서는 선언형 템플릿이나 프로비저닝 스크립트가 API를 호출해 자원을 생성·갱신한다. 실행 결과는 상태 파일로 추적되며 드리프트 감지가 가능하다.

IaC의 효과는 속도보다 일관성에서 더 크게 나온다. 개발, 스테이징, 운영 환경을 동일 패턴으로 재구성할 수 있어 장애 복구와 확장이 쉬워진다.
