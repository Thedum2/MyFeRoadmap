# Client Side Rendering

#개념

Client Side Rendering은 브라우저에서 JavaScript가 화면을 조립하는 렌더링 방식이다. 서버는 주로 데이터와 최소 셸을 제공하고, 최종 UI 구성 책임은 클라이언트가 가진다.

이 방식은 복잡한 상호작용 UI에 유리하다. 화면 상태와 이벤트가 클라이언트에 모여 있어 전환이 빠르고 사용자 반응성이 좋다.

low-level에서는 초기 HTML이 얇고, 스크립트 로드 후 데이터 요청과 컴포넌트 렌더링이 이어진다. JS 실행이 끝나기 전까지 표시 지연이 생길 수 있다.

한계는 초기 페인트 지연과 검색 엔진/미리보기 대응 부담이다. 그래서 SSR이나 정적 프리렌더링을 함께 조합하는 경우가 많다.
