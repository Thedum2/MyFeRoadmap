# URL

#개념

URL은 웹 자원의 위치와 접근 방법을 문자열로 표현하는 표준 형식이다. 쉽게 말해 "무엇을, 어디서, 어떤 방식으로 가져올지"를 한 줄로 담는다.

웹이 커지려면 자원을 식별하는 규칙이 먼저 필요했다. URL은 스킴, 호스트, 경로, 쿼리 같은 구성 요소를 통해 자원 주소 체계를 통일했다.

low-level에서는 브라우저가 URL을 파싱해 프로토콜과 목적지를 결정하고, 해당 프로토콜 규칙에 맞는 요청을 생성한다. 이 파싱 단계가 네트워크 요청의 출발점이다.

좋은 URL 설계는 단순한 주소 표기 이상이다. 캐시 전략, 라우팅 안정성, SEO, API 버전 관리까지 영향을 준다.
